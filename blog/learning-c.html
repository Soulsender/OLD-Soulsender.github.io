Learning C - Memory Management and Embedded Programming

Dec 12, 2022

The C programming language is something I've always wanted to learn.

To be honest, C scared me (and still scares me). I've heard too many stories about bad programming and overwriting memory with C. I could have decided to learn something like Rust instead, that doesn't have these problems as much, but C is far more common, and even though it is more than 50 years old now, it is still used everywhere.

Why did I want to learn C?

There is a couple different reasons. For one, C is used a lot in embedded programming. I had recently got a Flipper Zero, and I wanted to know more about electronics and embedded systems. I am aiming to create my own program on the flipper zero at some point, using the unleashed firmware as a starting point, and I will update this blog accordingly when I do.

C and C++ are also used a lot for malware development. I intend to learn C++ as well at some point, but this is lower on my priority list. Up to this point, I have only really learned interpreted languages such as Python, JavaScript, and Bash. While I know a tiny amount of Java from school, I will not be using Java. I hate Java.
